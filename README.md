# Customer Support Data Analysis

## Tools Used
Google Sheets, SQL, Power BI

## Project Overview
Analyzed customer support data to identify trends in resolution time, customer satisfaction, and support channel performance.

## Key Insights
- Technical issues required longer resolution time
- Chat channel was the most efficient
- Email support had lower customer satisfaction

## Dashboard Features
- KPI metrics (Resolution Time, Satisfaction, Ticket Count)
- Issue type analysis
- Channel performance comparison
- Ticket distribution visualization

## Outcome
Provided recommendations to improve customer support efficiency and customer satisfaction.
